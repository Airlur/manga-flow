# 漫译 MangaFlow - 产品需求文档 (PRD)

**版本**：1.0 | **更新日期**：2026-01-27

---

## 一、项目概述

### 1.1 产品定位

浏览器扩展插件，自动翻译漫画网页中的图片文字。对标「沉浸式翻译」漫画翻译模块，完全免费开源。

### 1.2 核心价值

| 对比项 | 沉浸式翻译 | 漫译 MangaFlow |
|--------|-----------|----------------|
| 价格 | 免费 100 张/月 | 完全免费 |
| 自定义 API | 不支持 | 支持 |
| 开源 | 否 | 是 |

---

## 二、功能需求

### 2.1 悬浮球交互

- **位置**：页面右下角，可拖拽
- **状态**：待机 / 翻译中 / 暂停 / 完成
- **操作**：点击切换翻译状态，长按打开设置

### 2.2 设置面板

| 设置项 | 类型 | 默认值 |
|--------|------|--------|
| 源语言 | 下拉 | 韩语 |
| 目标语言 | 下拉 | 简体中文 |
| 翻译引擎 | 下拉 | DeepL |
| API Key | 输入框 | 空 |
| 字体大小 | 滑块 | 14px |
| 字体颜色 | 颜色选择器 | #000000 |
| OCR 引擎 | 下拉 | 本地 |

### 2.3 核心翻译流程

```
图片检测 → 缓存检查 → OCR 识别 → 文本筛选 → 翻译 → 背景修复 → 译文渲染 → 缓存保存
```

### 2.4 批量处理策略

- 每批处理 3 张图片
- 处理完成后立即显示
- 范围：当前视口 + 下方 2 屏

### 2.5 缓存机制

- 缓存内容：图片指纹 + OCR 结果 + 翻译结果 + 渲染图片
- 有效期：30 天
- 容量限制：超过 500MB 自动清理

---

## 三、非功能需求

### 3.1 性能指标

- 单图处理时间 ≤ 8 秒
- 页面加载影响 < 100ms
- 内存占用 < 200MB

### 3.2 兼容性

- 浏览器：Chrome 88+ / Edge 88+
- 移动端：Android（Kiwi Browser）
- 站点：comix.to / toongod.org / omegascans.org / manhwaread.com

---

## 四、边界条件

### 4.1 支持的内容

- ✅ 气泡对话、旁白、心理描写

### 4.2 不支持的内容

- ❌ 拟声词、装饰文字、Logo、极小文字

### 4.3 错误处理

- OCR 失败 → 跳过该区域
- 翻译失败 → 尝试备选引擎
- 网络断开 → 暂停并提示

---

## 五、发布计划

### MVP 版本（v1.0）

- 悬浮球交互 + 设置面板
- Tesseract.js 本地 OCR
- DeepL/Google 翻译
- Canvas 背景修复 + 渲染
- LocalForage 缓存
- 4 个目标站点适配

- v1.2：更多翻译引擎 + 站点适配
