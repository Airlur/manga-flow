## v0.2 任务列表

### P0（必须先做）
- [x] OCR 裁剪识别：按候选 ROI 裁剪后调用 OCR（支持批量请求）
- [x] 文本块聚类：将同气泡/同段落行块合并为组（输出 group bbox）
- [x] 组级 mask + dilation（覆盖完整原文）
- [x] 组级渲染：译文按组排版，避免行级叠字

### P1（优化）
- [x] 轻量候选 ROI 检测（无模型版本：连通域/阈值法）
- [ ] 水印/广告关键词过滤（站点级可配置）
- [ ] OCR 结果二次修正：行间距、重叠率、相似文本去重
- [x] 开发模式开关（默认开启）：阶段选择 + 红框显示独立开关

  - 阶段 B：ROI + OCR（只识别裁剪区域）
  - 阶段 C：OCR + 翻译（不渲染）
  - 阶段 D：完整流程（擦除 + 渲染）
- [x] 缓存命中日志：OCR/翻译命中时在控制台明确标记
